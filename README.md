# uexWidgetOne
系统引擎

1、介绍

系统全局对象。WidgetOne应用管理器是整个Widget的运行环境，功能主要包括Widget管理、Widget内部窗口管理、Widget的生命周期管理、消息事件管理、Widget间通信机制和Widget的运行沙箱管理等。通过在UI层次上实现一套模拟系统级别的应用(Widget)及应用管理机制、窗口(Window)及多窗口管理机制，达到基于HTML5开发移动应用的目的，实现核心引擎的跨平台能力。
